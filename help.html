You must enter a name, email, and bar number in the settings sheet to run the script.

EXCLUDE CLOSED CASES
Change this setting to true if you want to ignore cases listed as closed on CourtConnect. You might also change this setting to true if you are experiencing errors.
This option is set to false by default because cases are often re-opened but remain listed as closed on CourtConnect.

RUN DOCKET MONITOR
The script will run automatically based on scheduled triggers, but you can also start the script from the menu (Docket Monitor > run docket monitor).

ERRORS
If you receive multiple emails notifying you of errors, try resetting the script (Docket Monitor > reset) and make sure you have the most recent version of the script. If the errors persist, delete the Docket Monitor folder and start over.
